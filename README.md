# Pace Buddy - The Pomodor Timer 🕰️
A productivity tool used to break work into 25-minute, high-focus intervals separated by 5-minutes short break and 18-minutes long breaks.
# Features ✨
* Modes - Switch between different intervals- focus, short and long breaks
* Timer - Updates every 1 second 
* Start/Pause - A button to start (green) or pause (red) the timer
* Reset button - Clean logic used to reset the timer and reverts back to the mode it is in
# TechStack ⚒️
* HTML5 - For the semantic structure
* Tailwind CSS - For a clean and interactive effects
* JavaScript - JS features like arrow func, template literals and ternary operator
* React.js - To build dynamic, state-driven user interface with timer, modes and active states without reloading the page
